Home

This is a website written by a student at San Francisco State University in the class CSC 412 (Advanced Software Lab). The purpose of this website is to demonstrate knowledge of the application of modern web development technologies such as HTML, Javascript (including jQuery) and PHP, SQL and their appliance in relation to databases so as to act as an interactive portfolio for any future employers.

This site uses all of the above-mentioned technologies except SQL for security reasons. Some responsive web design was also used in order to adapt to the high usage of mobile device to access web-pages. To see my current portfolio, see the Current section. To see interactive demonstrations of the above-mentioned web-technologies, see the Demonstrations section.
